# BookSeller

Краткое описание: Работаю над собственным проектом, представляющим собой распределенную систему с микросервисной архитектурой.

## Архитектура и технологии

### Основной стек
- **Бэкенд:** Микросервисы на Java Spring Boot
- **Service Discovery:** Spring Cloud Netflix Eureka Server
- **API Gateway:** Паттерн API Gateway для маршрутизации запросов
- **Event-Driven Architecture:** Apache Kafka для обработки событий
- **Документация:** Swagger для документирования API

### Авторизация и безопасность
- OAuth 2.0 аутентификация через Google и GitHub

### Базы данных и хранилища
- **Основное хранение данных:** PostgreSQL и MongoDB
- **Планы:** Поиск и логирование: Elasticsearch & Кэширование: Redis

### Тестирование
- Postman для ручного тестирования API
- **Планы:** Покрытие юнит-тестами более 90%

### Контейнеризация и оркестрация
- **Контейнеризация:** Docker
- **Планы:** Kubernetes для оркестрации (в разработке)

### Система контроля версий
- GitHub

## Запуск проекта

### Предварительные требования
- Java 11+
- Docker и Docker Compose
- Maven или Gradle
